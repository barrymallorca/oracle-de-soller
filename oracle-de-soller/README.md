# 🦜 L'Oracle de Sóller

**L'Oracle de Sóller** és un assistent d'informació cívica multilingüe per al municipi de Sóller, Mallorca.

*L'Oracle de Sóller is a multilingual civic AI assistant for the municipality of Sóller, Mallorca.*

*L'Oracle de Sóller es un asistente de información cívica multilingüe para el municipio de Sóller, Mallorca.*

---

## Sobre el projecte · About the project

Inspirat en el lloro de la **Faula** de Guillem de Torroella (1375) — el poema medieval en català en el qual el narrador troba un lloro sobre l'esquena d'una balena al Port de Sóller — L'Oracle respon preguntes sobre ordenances municipals, transport, farmàcies de guàrdia, emergències, agenda cultural i molt més, en qualsevol idioma.

Inspired by the parrot of the **Faula** by Guillem de Torroella (1375) — the medieval Catalan poem in which the narrator encounters a parrot on the back of a whale in the Port de Sóller — the Oracle answers questions about municipal bylaws, transport, duty pharmacies, emergencies, cultural events and much more, in any language.

## Característiques · Features

- 🌍 **Multilingüe** — Català, Español, English, Deutsch, Français, Italiano i més
- 💊 **Farmàcia de guàrdia** — Sap quina farmàcia fa guàrdia avui (cada dia de l'any)
- 📋 **Ordenances municipals** — Accés a totes les lleis municipals de Sóller
- 🚌 **Transport** — Horaris de tren, tramvia i autobús
- 🚨 **Emergències** — Telèfons d'emergència locals
- 💰 **Pressupost municipal** — Transparència total
- 🎭 **Agenda cultural** — Events i festivals

## Tecnologia · Technology

- **Frontend**: HTML/CSS/JavaScript — Cloudflare Pages
- **Backend**: Cloudflare Pages Functions
- **AI**: Anthropic Claude API
- **Domini**: oracledesoller.com
